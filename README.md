# 🚀 CodeHire AI

CodeHire AI is a modern placement preparation and coding progress tracking platform designed for students.

## 📌 Features

* 📊 Dashboard Overview
* 💻 DSA Progress Tracking
* 📚 Question Bank
* 📄 Resume Analyzer
* 🎤 Mock Interview Preparation
* 🗺️ Placement Roadmap
* 📈 Progress Tracker
* 🔥 Coding Streak System
* 🎯 Placement Readiness Tracker

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* VS Code
* Git & GitHub
